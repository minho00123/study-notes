# 컴파일러 언어(Compiler Language)
---
## Description
- 코드가 실행되기 전 단계인 컴파일 타임에 소스코드 전체를 한번에 머신 코드로 변환한 후 실행한다.
- 실행 파일을 생성한다.
- 컴파일 단계와 실행 단계가 분리되어 있다.
	- 명시적인 컴파일 단계를 거치고, 명시적으로 실행 파일을 실행한다.
- 실행에 앞서 컴파일은 단 한번 수행된다.
- 컴파일과 실행 단계가 분리되어 있으므로 코드 실행 속도가 빠르다.
