# Unmanaged Language
---
## Description
- 개발자가 명시적으로 메모리를 할당하고 해제하기 위해 `malloc()`과 `free()` 같은 저수준 메모리 제어 기능을 제공하는 프로그래밍 언어
- 메모리 제어를 개발자가 주도할 수 있으므로 개발자의 역량에 따라 최적의 성능을 확보할 수 있지만 그 반대의 경우 치명적 오류를 생산할 가능성도 있다.
