# Parsing
---
## Summary
> 정형화된 텍스트를 자료 구조(Data Structure)로 변환하는 과정이다.
## Description
- 텍스트를 읽고 메모리가 분석하기 쉽게 변환하여 텍스트를 변환하는 과정이다.
	- 보통 Tree형 자료 구조로 변환한다. Ex. HTML Parsing -> DOM Tree
