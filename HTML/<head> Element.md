# \<head\> Element
## Summary
> [[HTML(HyperText Markup Language) | HTML]] 문서의 정보를 나타낸다.
## Syntax
```html
<head></head>
```
## Description
- 웹 페이지의 보이지 않는 정보(= [[Metadata]])를 가진다.
	Ex. 웹 페이지의 제목, 설명, 사용할 파일 위치, 스타일(CSS)
