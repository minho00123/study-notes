# Viewport
---
## 📌 Description
- 컴퓨터 그래픽에서 현재 보고 있는 영역(일반적으로 직사각형)을 나타낸다.
- 웹 브라우저에서는 현재 보고 있는 문서의 창에 표시되는 부분을 의미한다.
- Viewport에서 현재 나타나고 있는 부분을 'Visual Viewport'라고 한다.
	- 웹 페이지를 확대할 경우 layout viewports는 같지만, visual viewport는 작아진다.
