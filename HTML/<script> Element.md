# \<script\> Element
---
## Description
- [[HTML(HyperText Markup Language)|HTML]]에서 실행 가능한 코드나 데이터를 삽입하는 데 사용된다.
- 일반적으로 JavaScript 코드를 삽입하거나 참조하는데 사용된다.
### Attributes
- `type`: 스크립트 종류를 나타낸다.
	- `module`: 코드가 JavaScript [[Module(모듈) | 모듈]]로 처리된다.
