# 토큰(Token)
---
## Description
- 문법적으로 더 이상 나눌 수 없는 코드의 기본 요소
- 여러 개의 토큰이 모여 [[문(Statement)]]을 형성한다.
## Examples
- [[키워드(Keyword)]], [[식별자(Identifier)]], [[연산자(Operators)]], [[리터럴(Literals)]], 세미콜론(;) 또는 마침표(.) 등과 같은 특수 기호
