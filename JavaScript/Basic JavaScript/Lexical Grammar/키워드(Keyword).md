# 키워드(Keyword)
---
## Summary
> 콘텐츠를 설명하는 단어 또는 구문
## Description
- 특별한 의미를 갖는 [[토큰(Token)]]이다.
- 자바스크립트 엔진은 키워드를 만나면 자신이 수행해야 할 약속된 동작을 수행한다.
	Ex. [[var]] 키워드를 만나면 자바스크립트 엔진은 뒤에 오는 변수 이름으로 새로운 변수를 선언한다.
