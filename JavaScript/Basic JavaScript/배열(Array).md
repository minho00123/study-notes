# 배열(Array)
---
## Description
- 하나의 변수명에 여러 요소들을 저장할 수 있는 모임
- 자바스크립트에서는 배열의 크기 조절이 자유롭고, 모든 데이터 타입을 요소로 가질 수 있다.
- Index는 0에서 시작하기 때문에 마지막 요소는 배열 길이에서 1을 뺀 값이다.
## Examples
```jsx
const arr = ['a', 'b', 'c', 'd'];
console.log(arr[2]) // c
console.log(arr.length) // 4
```
