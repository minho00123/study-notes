# 자바스크립트(JavaScript)
---
## 📌 Summary
> 웹페이지에서 동작하는 유일한 프로그래밍 언어
## 📌 Description
- JavaScript는 다음과 같은 특징을 가진다:
	- 명령형(imperative)
	- 함수형(functional)
	- Single-Threaded
	- 일급 함수(First-class Function)를 가진 인터프리터 프로그래밍 언어다.
	- 프로토타입 기반 객체지향 프로그래밍 언어
	- 멀티 패러다임 프로그래밍 언어
	- Dynamic 프로그래밍 언어
